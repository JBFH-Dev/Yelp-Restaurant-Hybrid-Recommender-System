----------README-----------
1. File Structure:
The files should be arranged as such:

hybrid_filt.py 
Data
|-- covid_19_dataset_2020_06_10
|   |--- yelp_academic_dataset_covid_filt.json
|    
|-- yelp_dataset
    |--- yelp_academic_dataset_business_filt.json
    |--- yelp_academic_dataset_review_filt.json

As set out in the submission folder.

2. Conda packages
Please set up your conda environment and install the following packages:
numpy
pandas
scipy
sklearn
tqdm



3. Running
This recommender system can be run by simply typing:
    python hybrid_filt.py 
into your conda environment.
